# ML Iris Project 

I know this is the simplest project/model but i'm uploading it as my first ML project/model.
Also it was made while learning in class.
Our professor instructed us how to build a simple classification model.
This model uses the Iris flower dataset to predict the species of an iris flower.

## Algorithm
  - Logistic Regression

## Language
  - Python

## Libraries
  - NumPy
  - Pandas
  - Scikit-learn

This model predicts the following iris flower species:-
- Setosa
- Versicolor
- Virginica
